# Hide and Seek

## By Shashank Prasanna (SpyGuy)

---

### How To Play:

- You are either the seeker or the hider
- Hiders
  - Have to avoid the seeker
  - Survive for 5 mins
- Seekers:
  - Have to find the hiders within 5 mins

### However, there are some obstacles

- Walls
- Hider Traps

### And Power-Ups

- Flashlights
- Running Shoes
